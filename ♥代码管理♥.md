# 加顿号

```js
  orderInfo?.orderNoAndCodeList.map((item: any, index: number) => (
                     <span key={item.name}>
                     <NavLink to={`/trading/order/detail?code=${item.value}`}>
                      {item.name}
                     </NavLink>
                     {index < orderInfo?.orderNoAndCodeList.length - 1 && '、'}
                                </span>
                            ))
```
